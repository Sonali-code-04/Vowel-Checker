# 🔤 Vowel Checker

## Description

A simple web application that checks whether a given character or text contains vowels.
This project demonstrates basic string handling and user input processing using JavaScript.


## Features

* 🔍 Check if a character is a vowel
* 📝 Accept user input
* ⚡ Instant result display
* 🎯 Simple and user-friendly interface


## Technologies Used

* HTML
* CSS
* JavaScript


## Learning Outcome

This project helped me understand:

* String manipulation in JavaScript
* Conditional logic
* Handling user input
* DOM manipulation


## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser



## Example

Input: `a` → Output: Vowel
Input: `b` → Output: Not a Vowel
